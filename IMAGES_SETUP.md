# Инструкция по добавлению изображений

Для полноценной работы приложения необходимо добавить следующие изображения в папку `assets/images/`:

## Требуемые файлы:

1. **banner.png** - Баннер "Здесь учатся герои" с изображением пожарного и спасателя
   - Сохраните первое изображение как: `assets/images/banner.png`

2. **emblem.png** - Эмблема колледжа (круглый логотип с каской, топором и книгой)
   - Сохраните второе изображение как: `assets/images/emblem.png`

## Где используются:

- **banner.png** - отображается на главном экране под карточкой специальности
- **emblem.png** - отображается в верхней части главного экрана как логотип колледжа

## После добавления изображений:

Приложение автоматически загрузит их при следующем запуске. Если изображения отсутствуют, будут показаны placeholder'ы с градиентными фонами.

## Команда для перезапуска (если нужно):

```bash
flutter run -d emulator-5554
```
